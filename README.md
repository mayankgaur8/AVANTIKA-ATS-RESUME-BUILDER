# ATS Resume Builder (React + Vite)

## Open in VS Code
1. Unzip this project
2. Open the folder in Visual Studio Code

## Run locally
```bash
npm install
npm run dev
```

## Export PDF (ATS-safe)
Click **Export PDF** → browser Print dialog → **Save as PDF**
(keeps text selectable for ATS)

## Notes
- Data is saved in LocalStorage (no login required).
- Includes basic ATS score + job description keyword matching.
